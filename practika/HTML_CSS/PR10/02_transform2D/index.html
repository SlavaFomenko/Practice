CSS Transform

translate (x, y)

translate

translate - зміна розташування елемента на значення X зліва на право і Y зверху вниз. Записується наступним чином: translate (x,y);

scale (x, y)

scale

scale - маштабує елемент, роблячи його більшим або меньшим. За замовченням значення дорівнює (1) - це реальний розмір елемента. Перше значення маштабує елемент по ширині, друге — по висоті. Відємне значення відображає елемент зеркально.

rotate (кут+deg)

rotate

rotate - повертає елемент на задану кількість градусів, відємне значення повертає елемент проти годиникової стрілки, плюсове — за годиниковою стрілкою.

skew (x-кут+deg, y-кут+deg)

skew

skew - деформує сторони елемента відносно кординат Х та У. При додатніх значеннях нахил здійснюється за годиниковою стрілкою, при відємних проти.

matrix (a, c, b, d, x, y)

matrix

matrix(a, c, b, d, x, y) - дозволяє обєднати декілька функцій трансформації в одну. В якості трансформації доступні поворот, маштабування, нахил, зміна положення.

Множинні 2D трансформації

translate

Так само, подібного можна досягти, використовуючи множинні трансформації. А саме записати кілька значень в одну властивість перетворення.

transform-origin

transform origin

transform-origin - дозволяє змістити центр трансформації, щодо якого відбувається зміна положення/розміру/форми елемента
